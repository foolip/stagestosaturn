About This eBook

This EPUB version of Stages to Saturn is based on the HTML version published by the NASA History Office. Many small changes and improvements have been made:

  • Hundreds of typos have been found and fixed by comparing the text to an independent scan of the book from the Internet Archive.
  • Many illustrations have been improved using high-resolution versions published by NASA and other sources.
  • Illustrations have been reorganized to better fit the EPUB medium.
  • The use of italic text, en dash (–) and em dash (—) has been restored to match the book.

A complete list of changes and the tools used can be found in the Git repository. This eBook was generated from commit REVISION.
